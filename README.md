# Nokiah_Snake-Game

A retro-style arcade snake game, build using Python and Pygame, that you can play on your mobile device or desktop.

Features:

 - Theme selection (Dark, Neon, Retro)
 - Difficulty level selection (Easy, Medium, Hard)
 - Gameplay screen:
     - Player_controlled snake
     - Food-target with movements
     - Snake length-expand effect
     - Current scores & high-scores
     - On-Screen buttons:
          - Move Left/Right
          - Move Up/Down
     - Game restart button


How to run:
 1. Install Python IDE
 2. Install Pygame
 3. Run the game! (code attached):
      ⇒ nokiah_snake_game.py
